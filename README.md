# PowerBI_Dashboard_Project-Amazon-Prime
# 📦 Amazon Sales & Operations Dashboard
# 🌟 Overview
This project features an interactive Power BI Dashboard designed to analyze Amazon sales performance, customer behavior, and operational efficiency. 📈 The dashboard provides a "bird's-eye view" of the business, allowing stakeholders to track revenue trends, identify top-selling products, and monitor shipping performance across different regions. 🌎

# ✨ Features
Sales Performance Tracking: Comprehensive visuals showing Total Sales, Profit Margins, and Order Quantities over time. 💰

Product Hierarchy Analysis: Breakdown of performance by Category and Sub-Category to identify growth opportunities. 🍎

Geographic Insights: Interactive maps illustrating sales distribution and regional demand. 🗺️

Customer Segmentation: Analysis of order patterns to understand high-value customer groups. 👤

Operational Metrics: Tracking of shipping status (e.g., Shipped, Cancelled, Pending) to evaluate fulfillment health. 🚚

# 🛠️ Technologies Used
Power BI Desktop: For end-to-end data visualization and report design. 🖥️

Power Query: To handle data cleaning, such as removing duplicates and formatting date columns. 🧹

DAX (Data Analysis Expressions): To calculate complex measures like Year-to-Date (YTD) Sales or Profit percentage. 🔢

# 📊 Key Functionalities
Dynamic Slicers: Users can filter the entire report by State, Category, or Time Period with a single click.

Cross-Highlighting: Selecting a specific product category automatically updates all other visuals to show related data.

Trend Forecasting: Identifying seasonal peaks in sales to help with inventory planning.
